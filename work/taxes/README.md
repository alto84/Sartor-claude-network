- EIN for Solar Inference LLC (39-4199284) is acceptable as it is a business EIN, not personal.
- No personal SSNs in any file.

### What to check during a tax status review
1. Are all filing deadlines tracked and current?
2. Has Alton sent required documents to Jonathan Francis?
3. Is the personal 1040 filed or extended?
4. Is the Form 990 for Sante Total on track?
5. Is the solar roof on track to be in service before July 4, 2026?

### What requires human confirmation
- Filing any return or extension
- Contacting the CPA on Alton's behalf
- Interpreting ambiguous tax code — escalate to CPA
- Any payment of taxes owed

### Use current tax code only
Do not reference deprecated rules. ITC is under current IRC §48, bonus depreciation under §168(k) (OBBB/TCJA provisions). NJ SuSI is a state program — verify current rates with CPA.
