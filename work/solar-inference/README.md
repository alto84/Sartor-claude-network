# Solar Inference LLC

Solar Inference LLC is a New Jersey multi-member LLC (Alton 50%, Aneeta 50%) operating a solar-powered GPU inference business. The business model: a $438,829 Tesla Solar Roof generates electricity to power a GPU server (RTX 5090) that rents compute on vast.ai. Currently pre-revenue. Critical path: solar roof installation must complete before July 4, 2026 to claim $529K in tax benefits. Priority level: HIGH.
