# Examples

This folder contains one reference page showing what a real wiki page looks like after you've filled it out. It's here so you can see the expected format, frontmatter density, and callout usage.

## Files

- `CRS-example.md` — a full adverse event page for Cytokine Release Syndrome. Shows frontmatter with all fields filled in, multiple wikilinks to related mechanism/drug/signal pages, a callout with a published reference, and a history section.

## How to use

1. Read the example to get a feel for the target density and structure
2. When creating your first real page, copy the matching template from `templates/` (not this example)
3. The example is deliberately "complete-looking" — your first pages can be much smaller and still be useful. The wiki compounds over time; don't try to front-load everything on day one.

## What to copy from the example

- **Frontmatter density:** required fields filled, optional fields used where they add value
- **Wikilink density:** 5-10 per page is a good target for core entity pages
- **Callout usage:** `[!fact]` for verified claims with citations, `[!warning]` for pending label updates or contradictions
- **Section ordering:** Definition → Mechanism → Affected classes → Management → Signal history → Labeling → References → History

## What NOT to copy

- The specific CRS content. It's illustrative, not authoritative. Use your own authoritative sources.
- The exact list of related pages. Your wiki's cross-reference graph will look different.
