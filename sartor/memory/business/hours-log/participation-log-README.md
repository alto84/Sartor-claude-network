# Participation log — Solar Inference LLC (§469 material participation)

Manual companion to the automated `all-hours.csv` (which captures only Alton's
Claude-session keyboard time). This log captures what the auto-logger cannot:
physical work, site visits, contractor meetings, phone calls — for BOTH spouses.

**Why both spouses:** §469(h)(5) / Treas. Reg. §1.469-5T(f)(3) — a spouse's
participation is attributed to the taxpayer. Alton's and Aneeta's hours COMBINE
for every material-participation test. Aneeta's hours help; they do not compete.
(They do not, however, create §704(d) basis for her — separate matter.)

## Rules for entries

1. **Log contemporaneously** — same day or same week. One line per work session.
2. **Operator-level descriptions**: what was physically done or decided
   ("assembled rtxserver: GPU seating, cabling", "met mason re solar prep").
   Avoid investor-grade phrasing ("reviewed dashboard", "discussed business") —
   §1.469-5T(f)(2)(ii) excludes investor activity.
3. **Corroboration column**: calendar event, email thread, receipt, photo —
   something a third party could check the entry against.
4. **Backfilled rows** are marked `2026-06-09 backfill` in `prepared` and were
   seeded from dated records (calendar/Gmail/receipts). Fill `person` (Alton /
   Aneeta / both — use two rows if both) and `hours` from memory NOW while
   fresh; do not guess beyond what you can defend. Rows going forward get
   `prepared` = the date written.
5. Categories: `assembly`, `contractor`, `planning`, `ops` (server/listing
   admin), `site` (install supervision). Household/family time and W-2-job
   time never go in this log.

## Year-end use

At TY2026 close: sum per person, combine for the §469 tests (100-hr (a)(3) is
the target; 500-hr (a)(1) if the install year runs hot), and file the log +
corroboration in the CPA package. See matters/469-material-participation.md.
