# Matter: Sante Total Form 990-N → 990-EZ migration

## Issue

Sante Total's gross receipts approaching the $50K 990-N threshold. Plan migration to Form 990-EZ for TY2026 forward.

## Facts

## Authority

- **IRC §6033**: Annual return filing requirement for exempt organizations.
- **Treas. Reg. §1.6033-2(g)(1)(iii)**: Filing thresholds — 990-N for under $50K gross receipts; 990-EZ for $50K-$200K; 990 (full) for $200K+.
- **Form 990-EZ**: Instructions and schedules.

## Analysis

If Sante Total's TY2026 gross receipts cross $50K: 990-EZ required. If they stay under: 990-N continues to work.

Migration to 990-EZ is straightforward but requires:
- Detailed financial records (not just gross receipts)
- Schedule O for narrative
- Schedule A for public charity status
- More disclosure (officers, key employees, top contractors, etc.)

If Alton handles 990-N himself currently, migrating to 990-EZ might warrant a CPA or specialized nonprofit preparer (NOT in JF's scope per [[TAXES]]).
