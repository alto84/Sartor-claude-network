## How to read this

- **path** = `light` (direct tmux send) or `heavy` (inbox file + git commit + push)
- **slug** for heavy sends shows the first 8 chars of the commit SHA in parens — find the directive at `sartor/memory/inbox/{peer}/{ts}-{slug}.md`
- **ack** = ✓ when capture-pane after 4s showed a processing indicator (Cogitating/Forming/tool call); — when no indicator visible (peer may still be working, but the send isn't confirmed)
- **flags** = §7 content-judgment hits (pricing change / money / medical / kid PII patterns); the script does NOT block on flags, peer can decline

Full per-send detail is in `sartor/memory/peer-sends/{YYYY-MM}.jsonl`.
