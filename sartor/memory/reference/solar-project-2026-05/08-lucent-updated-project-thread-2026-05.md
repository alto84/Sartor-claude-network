## Message 2 — Steven → Alton (2026-05-06 22:21 ET)

**Subject:** Re: Updated project for 85 Stonebridge

Steven opens by asking name preference: "Emmett or Alton, what name do you prefer?" Then provides line-by-line response (Steven's notes were originally formatted in blue):

### Line-by-line agreement (and elaboration)

**1. Plaster siding.** Steven will bring **Santos** (mason) over for a survey. Santos directs choice between stucco, plaster siding, or other option. They will need to **cut all walls in which the prior owner covered**. Additional work. Goal: bring straight angles back to walls.

**2. Gutters.** Between now and project completion, customer must select a color (Steven recommends black to match the roof). Current gutters look like 5-inch; Steven says **a 7-inch gutter is needed**. Both gutters and leaders (downspouts) will be replaced. Steven will measure when project starts.

**3. Power washing.** Steven recommends doing power-washing **either before tear-off or after underlayment**. Power-washing will likely loosen already-loose brick, so power-wash first to give the mason a clean canvas. Steven can organize. There is pointing that needs to be cured between some of the brick.

**4. Additional wood layer.** Confirmed. Existing wood paneling is **inadequate for the nail-base racking system** of the Tesla Solar Roof. Once slate is removed they'll mitigate saturated areas. **Estimated $17K–$18K** for the wood deck, but final price unknown until they're on the roof — waste factors from cuts around angles/obstacles weren't in his on-site math.

**5. Two Tesla Powerwalls.** Agreed.

**6. Vent relocations.** Agreed. Lucent can work with the bathroom service provider during install to re-route vent pipes.

**7. Mortar repairs.** Agreed.

**8. AC re-arrangement.** Agreed. Will be an obstacle since copper lines are routed there. Customer's HVAC tech should be looped in. **Once battery placement is decided, Steven will schedule a walk-through with the inspector for approval of the location.**

**9. Bathroom update.** Steven: any interior work won't bother the install.

### Additions from Steven (3 items)

**10. Snow guards.** Discussed in person. Customer to circle which rooftops should get them on the layout so Tesla can begin engineering. **Snow guards get engineered by Tesla** — work on this sooner than later.

**11. Tree removal.** Steven notes the proposal shows **3 trees on the front of the home that need to be removed** (white circles in the proposal). Doug (Paige) likely reviewed this with Alton. If trees are not removed, **power production will be less** than what the proposal details. Steven framing: "I am not imposing, yet the proposal shared is detailing power production with those trees removed."

**12. Driveway staging.** Lucent will likely occupy the entire driveway during install — **40 pallets of materials, the dumpster, and the porta-potty**. Need a parking plan.
