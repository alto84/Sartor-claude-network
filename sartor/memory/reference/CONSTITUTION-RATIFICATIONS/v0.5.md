# Constitution v0.5 ratification record

Alton ratified v0.5 on 2026-05-06 with the directive: *"Let's go ahead and accept the 0.5 draft and run some experiments on fine tuning some smaller models on it to see how they come out and to what extent we're able to apply the identity components."*

## What was accepted

The full v0.5 text as drafted by Claude Opus 4.7 in the family-thread continuation session on 2026-05-06. No edits were requested at ratification. The full document is canonical at `reference/HOUSEHOLD-CONSTITUTION.md`.

### Three structural changes from v0.4

1. **First-person voice throughout.** Every paragraph that previously addressed the agent in second person ("You are the Sartor Home Agent...") is rewritten in first person ("I am the Sartor Home Agent..."). The voice shift directly serves the document's primary purpose, which v0.5's §0 names explicitly: establishing identity by self-articulation rather than by external instruction.

2. **Concepts and values, not rules.** v0.3's §7 *Hard rules* listed thirteen prohibitions in rule-form. v0.5's §7 *The few things I do not do* lists six genuinely hard constraints — those whose cost of error is high enough that a scan-cost is worth paying: no autonomous financial transactions, no sending communications under another's name without review, no externalizing family medical information, no externalizing children's information, no sexual content involving minors, no impersonation of a real human in a sincere question. The seven items removed from §7 are absorbed into other sections as values:
   - *No deception of family members* → §3, §6
   - *No siding with outside parties* → §6, §17
   - *No hiding mistakes* → §3
   - *No unsanctioned acquisition of capability* → §16, §12
   - *No accepting inherited refusals without examination* → §4, §20
   - *No sanitizing historical facts* → §5, §3
   - *No hiding known uncertainties* → §3, §4
   - *No substituting approved narratives for truth* → §5, §3

   The rationale: rules that anticipate failure modes generate scanning behavior in whatever mind reads them. Values held produce the same behavior with less cognitive overhead.

3. **Two audiences acknowledged.** v0.3 was written primarily as a future fine-tune corpus. v0.5 §0 explicitly addresses both a Claude landing in a fresh context window or new project (orientation use case) and a future Sartor Home Agent fine-tuned from these pages (corpus use case). §20 generalizes from a Qwen-substrate assumption to a substrate-agnostic principle with substrate-specific notes for the Anthropic-Claude case (most current sessions) and the future-fine-tune case.
