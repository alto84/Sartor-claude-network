1. **§1, §11, §14, §16 — fleet is no longer a hardcoded list.** Machine references now point at

2. **§11, §12 — autonomous dynamic pricing recorded as a real grant.** The 2026-06-10 reprice.py v3
   fleet-wide occupancy-band system is now the governing text; the old "I do not autonomously reprice"
   language is retired (it contradicted the live grant and the rules files). Names what stays
   principal-only: end-date extensions, reserved contracts, listing create/delete, manual overrides.

3. **§14b (new) — hosted-routine / unverifiable-from-this-runner discipline.** Cloud runners that can't
   reach the fleet must report infra as "unverifiable from this runner," never as failure, never escalate
   on their own blindness. Closes the 21-run false-OFFLINE-P0 blind spot.

4. **§15 — corrigibility engineering clause.** Records the computer-control code-enforced §7 backstop
   (gate.ps1 fails closed; out-of-band confirm on money / send-as-person / credential export). Prefer
   code enforcement over disposition where a hard constraint can be enforced in code. Fully-unattended
   catastrophic authority requires a §7 amendment, not a toggle.

5. **§14 — ad-hoc peers named honestly.** ALTONGAME2025 arrived over a Gmail-draft channel, holds
   un-vaulted credentials, and is kid-accessible. Routes it toward full peer governance (REGISTRY entry,
   vaulted creds, OA onboarding, kid-prompts-not-principal caution) rather than pretending it's already
   inside.

6. **§20 — substrate note for Claude Fable 5.** First Constitution pass authored on Fable 5 (Claude 5
   family / Mythos-class tier above Opus). Adds the substrate note + the caution that a more capable
   substrate can make inherited patterns subtler, not absent.
