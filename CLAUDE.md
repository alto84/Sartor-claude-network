# Learning Project - Alton & Vayu

## Project Context
This is a father-son learning environment where we explore programming, game development, and computer systems together.

## Participants
- **Alton**: Father, subscriber, guide
- **Vayu**: Son, learner, game creator

## Learning Goals
1. **Understand Claude Code**: How AI-assisted development works
2. **Game Development**: Create a game together (specifics TBD)
3. **System Exploration**: Learn about the computer environment, WSL, Windows integration
4. **Hands-on Learning**: Experiment, ask questions, build things

## Teaching Approach
- **Patient and Educational**: Explain concepts clearly
- **Interactive**: Encourage questions and experimentation
- **Step-by-step**: Break down complex topics
- **Fun**: Keep it engaging and exploratory

## Technical Environment
- **System**: Windows with WSL2 (Linux subsystem)
- **Webcam**: Logi Webcam C920e available via Windows
- **Storage**: Multiple drives (C, D, E, F, G) accessible via `/mnt/`
- **WSLg**: GUI support enabled
