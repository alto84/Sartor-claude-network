┌──────────────────────────────────────────────────────────────────────┐
│ WHAT IT DOES                                                         │
└──────────────────────────────────────────────────────────────────────┘
  • Injects roadmap context into every agent session
  • Tracks task completion automatically
  • Accumulates learning data for self-improvement
  • Provides agents with current phase, progress, and next tasks

┌──────────────────────────────────────────────────────────────────────┐
│ HOW IT WORKS                                                         │
└──────────────────────────────────────────────────────────────────────┘
  Session Start → inject-roadmap.sh → Agent sees context
         ↓
  Agent works (Write/Edit/Bash)
         ↓
  PostToolUse → record-completion.sh → Logs learning data

┌──────────────────────────────────────────────────────────────────────┐
│ QUICK COMMANDS                                                       │
└──────────────────────────────────────────────────────────────────────┘
  View current phase:
    .claude/hooks/inject-roadmap.sh 2>&1

  Run demonstration:
    .claude/hooks/test-hooks-demo.sh

  View task completions:
    cat .claude/.task-completion-log

  Analyze learnings:
    cat .claude/.learnings.jsonl | jq .

  Count completions by tool:
    cat .claude/.learnings.jsonl | jq -r '.tool' | sort | uniq -c

  Set manual phase (0-5):
    echo "2" > .claude/.roadmap-state

  Return to automatic:
    rm .claude/.roadmap-state
